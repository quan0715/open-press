# Social Post Design

This starter is a fixed square page system. The document size comes from
`config.page = "social-square"` and resolves to `1080px x 1080px`.

Use it for short, direct publishing assets:

- one idea per page;
- strong title;
- compact supporting copy;
- optional small footer metadata.

Keep each page readable at thumbnail size. Avoid dense tables, long footnotes,
or multi-column body text.
