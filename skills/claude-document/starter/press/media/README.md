# media/

放這份文件用到的所有圖片、icon、品牌 asset。Markdown 內以 `media/<file>` 引用，會被解析為 `document/media/<file>`。

建議檔名規則：

- 截圖：`<topic>-<screen>.png`
- 場域 / 情境照片：`<topic>-<scene>.jpeg`
- 品牌標誌：`logo-<name>.svg` 或 `logo-<name>.png`
- 概念示意圖：`concept-<name>.png` / `concept-<name>.svg`
- 人物照：`person-<name>.jpg`

統一用小寫 + 連字號；非必要避免空白與大寫。
