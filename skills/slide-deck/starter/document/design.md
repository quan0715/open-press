# Slide Deck Design

This starter is a fixed 16:9 page system. The document size comes from
`config.page = "slide-16-9"` and resolves to `1920px x 1080px`.

The base rule is one idea per slide:

- short title;
- one supporting block;
- optional list of 3 to 5 points.

Do not make the slide respond to viewport width. Let the workbench scale the
whole page.
