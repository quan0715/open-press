本文件說明如何在 OpenPress 中安裝與運用各種 Skills。

## 步驟 1：了解可用的 Skills

OpenPress 提供多種不同類型的 Skills，以滿足不同工作流程的需求：

- **系統操作：** `openpress`（CLI 使用、驗證、匯出、路由）、`openpress-upgrade`（套件升級與 Migration QA）、`openpress-deploy`（部署）。
- **文件建立：** `openpress-create-pages`（A4 等頁面文件）、`openpress-create-slide`（簡報文件）。
- **審閱：** `/apply-comments`（處理行內註解）。
- **外部 Starter：** 專案需要具體範例或可重用起始檔時，可另外安裝 starter-bearing skills。

## 步驟 2：安裝外部或創意 Skills

若要更新既有 workspace 內建的 OpenPress skills，請執行：

```bash
npm run openpress:skills
# 或在只安裝 @open-press/core 的 workspace：
node node_modules/@open-press/core/engine/cli.mjs skills:sync .
```

若您需要特殊的設計或模板（例如社交卡片），您可以透過 npx 安裝外部 skill。例如，安裝社交卡片 skill：

```bash
npx -y skills@latest add quan0715/openpress-social-card-skill
```

## 步驟 3：透過 Agent 應用 Skill

安裝完畢後，請引導您的 Agent 使用該 skill：

1. 指示 Agent 讀取該 skill 的 `SKILL.md` 說明。
2. 讓 Agent 複製或改編 skill 所提供的 `starter/` 範例檔案到您的 OpenPress Workspace 中。
3. 如果是頁面或簡報建立的 skill，請 Agent 根據官方說明產生相對應的 `press.tsx` 與目錄結構。

## 步驟 4：驗證 Skill 產出

無論您使用了哪種 Skill 進行結構的建立或內容的引入，請務必在結束前執行建置來確認專案的健康度：

```bash
npm run build
```
