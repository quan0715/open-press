## SKILL catalog changes

Two skills were folded into existing ones:

- `openpress-update` → folded into `openpress` (new "Updating An Existing Workspace" section).
- `openpress-document-hierarchy` → folded into `openpress-writing` (new "Hierarchy" section).

`npx skills upgrade` removes the stale `.agents/skills/openpress-update/` and `.agents/skills/openpress-document-hierarchy/` directories. No further action needed.

If you have user-authored SKILL files that mention either name, update them to point at the new parent:

- `Find`: `rg 'openpress-update|openpress-document-hierarchy' .agents/skills/`
- `Replace`: see the table below

| Old reference | New target |
| --- | --- |
| `use openpress-update skill` | `use openpress skill` (look at the "Updating An Existing Workspace" section) |
